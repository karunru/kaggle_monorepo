### 1. データセット概要
- **train.csv**: 574,945行 × 341列

### 2. train.csv文字列カラムのユニーク値
- **sequence_type**: 2値（Target, Non-Target）
- **orientation**: 4値（体位）
- **behavior**: 4値（行動フェーズ）
- **phase**: 2値（Gesture, Transition）
- **gesture**: 18値（ジェスチャー種類）
  - Target（BFRB様）: 8種類
  - Non-Target: 10種類

### 3. 共起関係
- ジェスチャーと体位の強い関係性（特定体位でのみ実行可能なジェスチャー）
- フェーズと行動の明確な対応関係
- 全被験者が両タイプのシーケンスを実行

### 4. センサーデータ構成
- IMU: 7列（acc_xyz, rot_wxyz）
- Thermopile: 5列（thm_1-5）
- Time-of-Flight: 320列（tof_1-5_v0-63）
